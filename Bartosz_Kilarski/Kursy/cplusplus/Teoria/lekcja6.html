1. Używanie przestrzeni nazw (namespace) do organizacji kodu

Przestrzenie nazw służą do organizowania kodu i unikania konfliktów nazw. Oto przykład użycia przestrzeni nazw:

#include <iostream>

using namespace std;

namespace Matematyka {
    int dodaj(int a, int b) {
        return a + b;
    }
}

namespace Geometria {
    void rysuj() {
        cout << "Rysuję figurę" << endl;
    }
}

int main() {
    cout << "Wynik dodawania: " << Matematyka::dodaj(5, 3) << endl;
    Geometria::rysuj();

    return 0;
}

2. Definiowanie i używanie modułów: pliki nagłówkowe i źródłowe

Moduły w C++ dzielą kod na pliki nagłówkowe (.h) i źródłowe (.cpp). Plik nagłówkowy zawiera deklaracje, a plik źródłowy definicje funkcji i metod. Przykład:

// Plik: matematyka.h
#ifndef MATEMATYKA_H
#define MATEMATYKA_H

int dodaj(int a, int b);

#endif

// Plik: matematyka.cpp
#include "matematyka.h"

int dodaj(int a, int b) {
    return a + b;
}

// Plik: main.cpp
#include <iostream>
#include "matematyka.h"

using namespace std;

int main() {
    cout << "Wynik dodawania: " << dodaj(5, 3) << endl;

    return 0;
}

3. Importowanie bibliotek standardowych i zewnętrznych

Biblioteki standardowe są dostarczane z kompilatorem, natomiast biblioteki zewnętrzne muszą być dołączane osobno. Oto przykład importowania bibliotek standardowych i zewnętrznych:

#include <iostream>
#include <vector>
// #include <mojaBiblioteka.h> // Przykład dołączenia zewnętrznej biblioteki

using namespace std;

int main() {
    vector<int> liczby = {1, 2, 3, 4, 5};
    for (int liczba : liczby) {
        cout << liczba << " ";
    }
    cout << endl;

    return 0;
}

4. Tworzenie i używanie własnych bibliotek

Własne biblioteki można tworzyć, aby ponownie wykorzystywać kod w różnych projektach. Oto przykład tworzenia i używania własnej biblioteki:

// Plik: mojaBiblioteka.h
#ifndef MOJA_BIBLIOTEKA_H
#define MOJA_BIBLIOTEKA_H

void przywitaj();

#endif

// Plik: mojaBiblioteka.cpp
#include <iostream>
#include "mojaBiblioteka.h"

using namespace std;

void przywitaj() {
    cout << "Witaj z mojej biblioteki!" << endl;
}

// Plik: main.cpp
#include <iostream>
#include "mojaBiblioteka.h"

using namespace std;

int main() {
    przywitaj();

    return 0;
}
