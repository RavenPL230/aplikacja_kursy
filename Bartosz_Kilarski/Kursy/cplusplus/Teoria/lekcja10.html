1. Wprowadzenie do zarządzania pamięcią: podstawowe pojęcia

Zarządzanie pamięcią w C++ jest kluczowe dla efektywnego działania aplikacji. Obejmuje to zrozumienie podstawowych pojęć, takich jak wskaźniki, referencje, dynamiczna alokacja pamięci oraz smart wskaźniki. Oto główne aspekty:

  • Wskaźniki i referencje: Mechanizmy pozwalające na manipulowanie adresami pamięci oraz aliasami zmiennych.
  • Dynamiczna alokacja pamięci: Przydzielanie pamięci w trakcie działania programu.
  • Smart wskaźniki: Wskaźniki, które zarządzają pamięcią automatycznie, co zmniejsza ryzyko wycieków pamięci.
  • Mechanizmy C++11: Nowe funkcje i wskaźniki wprowadzone w C++11, które ułatwiają zarządzanie pamięcią.

2. Wskaźniki i referencje: definicja, użycie i różnice

Wskaźniki są zmiennymi przechowującymi adresy innych zmiennych, natomiast referencje są aliasami dla istniejących zmiennych. Oto przykłady użycia:

#include <iostream>

using namespace std;

int main() {
    int a = 10;
    int* ptr = &a; // wskaźnik
    int& ref = a;  // referencja

    cout << "Wartość a: " << a << endl;
    cout << "Wartość przez wskaźnik: " << *ptr << endl;
    cout << "Wartość przez referencję: " << ref << endl;

    return 0;
}

3. Dynamiczna alokacja pamięci: new, delete

Dynamiczna alokacja pamięci pozwala na przydzielanie pamięci w trakcie działania programu. Używamy do tego operatorów new i delete. Przykład:

#include <iostream>

using namespace std;

int main() {
    int* ptr = new int(5); // alokacja pamięci
    cout << "Wartość: " << *ptr << endl;
    delete ptr; // zwolnienie pamięci

    return 0;
}

4. Smart wskaźniki: unique_ptr, shared_ptr, weak_ptr

Smart wskaźniki zarządzają pamięcią automatycznie, co pomaga uniknąć wycieków pamięci. Oto przykłady:

#include <iostream>
#include <memory>

using namespace std;

int main() {

    unique_ptr<int> uniqPtr(new int(10));
    cout << "Wartość unique_ptr: " << *uniqPtr << endl;

    shared_ptr<int> sharPtr(new int(20));
    cout << "Wartość shared_ptr: " << *sharPtr << endl;

    weak_ptr<int> weakPtr = sharPtr;
    if (auto spt = weakPtr.lock()) {
        cout << "Wartość weak_ptr: " << *spt << endl;
    } else {
        cout << "Obiekt nie istnieje" << endl;
    }

    return 0;
}

5. Mechanizmy zarządzania pamięcią w C++11 i późniejszych wersjach

C++11 wprowadził nowe mechanizmy zarządzania pamięcią, takie jak smart wskaźniki oraz funkcje pomocnicze. Przykład użycia:

#include <iostream>
#include <memory>

using namespace std;

int main() {
    // Przykład użycia unique_ptr bez make_unique
    unique_ptr<int> ptr(new int(30));
    cout << "Wartość unique_ptr: " << *ptr << endl;

    return 0;
}
