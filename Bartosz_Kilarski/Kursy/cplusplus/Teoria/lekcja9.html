1. Funkcje anonimowe (lambda)

Funkcje anonimowe, czyli lambda, są funkcjami bez nazwy. Można je używać w miejscach, gdzie potrzebna jest funkcja jako argument. Oto przykład użycia lambda:

#include <iostream>
#include <algorithm>
#include <vector>

using namespace std;

int main() {
    vector<int> liczby = {1, 2, 3, 4, 5};

    // Funkcja lambda do wypisywania elementów
    auto wypisz = [](int x) {
        cout << x << " ";
    };

    for_each(liczby.begin(), liczby.end(), wypisz);
    cout << endl;

    return 0;
}

2. function i bind

function to elastyczny sposób przechowywania funkcji, a bind umożliwia tworzenie nowych funkcji z istniejących poprzez wiązanie argumentów. Przykład:

#include <iostream>
#include <functional>

using namespace std;

void dodaj(int a, int b) {
    cout << "Wynik dodawania: " << a + b << endl;
}

int main() {
    // Przechowywanie funkcji w function
    function<void(int, int)> func = dodaj;

    // Użycie funkcji
    func(5, 3);

    // Wiązanie argumentu do funkcji
    auto dodaj5 = bind(dodaj, 5, placeholders::_1);
    dodaj5(10);

    return 0;
}

3. Algorytmy STL: transform(), for_each()

Algorytmy STL ułatwiają manipulację kontenerami. Oto przykłady użycia transform() i for_each():

#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

int main() {
    vector<int> liczby = {1, 2, 3, 4, 5};
    vector<int> kwadraty(liczby.size());

    // Funkcja lambda do obliczania kwadratów
    transform(liczby.begin(), liczby.end(), kwadraty.begin(), [](int x) { return x * x; });

    // Funkcja lambda do wypisywania elementów
    for_each(kwadraty.begin(), kwadraty.end(), [](int x) { cout << x << " "; });
    cout << endl;

    return 0;
}
