• Kursy C++

1. Co to jest C++?

C++ to język programowania ogólnego przeznaczenia, oparty na C, który wspiera programowanie zorientowane obiektowo. Jest używany w wielu dziedzinach, takich jak tworzenie oprogramowania systemowego, gier, oraz aplikacji wymagających wysokiej wydajności.

2. Instalacja C++

Aby rozpocząć programowanie w C++, należy zainstalować odpowiednie środowisko programistyczne, takie jak Visual Studio lub Code::Blocks. Dostępne są również kompilatory C++ do użytku w przeglądarce np: OnlineGDB.

3. Pierwszy program w C++

Po zainstalowaniu środowiska można napisać swój pierwszy program w C++. Otwórz edytor kodu i wpisz:

#include <iostream>
using namespace std;

int main() {
    cout << "Witaj, świecie!" << endl;
    return 0;
}

Po skompilowaniu i uruchomieniu programu powinien on wyświetlić tekst: Witaj, świecie!

4. Podstawowe typy danych w C++

  • Liczby całkowite (int, short, long):
    • int: Liczby całkowite o standardowym zakresie. Przykład:
      int x = 5;
      int y = -3;
    • short: Krótszy typ całkowity, zajmuje mniej pamięci. Przykład:
      short s = 32767;
    • long: Dłuższy typ całkowity o większym zakresie. Przykład:
      long l = 9223372036854775807L;
  • Liczby zmiennoprzecinkowe (float, double):
    • float: Liczby zmiennoprzecinkowe z pojedynczą precyzją. Przykład:
      float a = 3.14f;
    • double: Liczby zmiennoprzecinkowe z podwójną precyzją. Przykład:
      double b = -0.001;
  • Znaki (char):
    • char: Pojedyncze znaki. Przykład:
      char letter = 'A';
  • Napisy (std::string):
    • string: Ciągi znaków tekstowych. Przykład:
      std::string text = "Hello";
  • Typ logiczny (bool):
    • bool: Typ logiczny reprezentujący wartości prawdy (true) i fałszu (false). Przykład:
      bool is_valid = true;
  • Typ void:
    • void: Specjalny typ używany w funkcjach, które nie zwracają wartości. Przykład:
      void function_name();

5. Operatory matematyczne i logiczne

  • Operatory matematyczne:
    • Dodawanie (+): Dodaje dwie liczby. Przykład:
      int sum = 5 + 3;
    • Odejmowanie (-): Odejmuje jedną liczbę od drugiej. Przykład:
      int difference = 10 - 4;
    • Mnożenie (*): Mnoży dwie liczby. Przykład:
      int product = 7 * 6;
    • Dzielenie (/): Dzieli jedną liczbę przez drugą. Przykład:
      double quotient = 8 / 2;
    • Reszta z dzielenia (%): Zwraca resztę z dzielenia. Przykład:
      int remainder = 10 % 3;
    • Inkrementacja (++): Zwiększa wartość zmiennej o 1. Przykład:
      int x = 5; x++;
    • Dekrementacja (--): Zmniejsza wartość zmiennej o 1. Przykład:
      int x = 5; x--;
  • Operatory logiczne:
    • AND (&&): Zwraca 1, jeśli oba warunki są prawdziwe. Przykład:
      bool result = (5 > 3) && (8 > 6);
    • OR (||): Zwraca 1, jeśli przynajmniej jeden z warunków jest prawdziwy. Przykład:
      bool result = (5 > 3) || (8 < 6);
    • NOT (!): Zwraca 1, jeśli warunek jest fałszywy, inaczej 0. Przykład:
      bool result = !(5 > 3);
    • Równoważność (==): Sprawdza, czy dwie wartości są równe. Zwraca 1 (prawda) lub 0 (fałsz). Przykład:
      bool isEqual = (5 == 5);
    • Różność (!=): Sprawdza, czy dwie wartości są różne. Zwraca 1 (prawda) lub 0 (fałsz). Przykład:
      bool isNotEqual = (5 != 3);
