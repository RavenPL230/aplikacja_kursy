    1. Tablice:

    Tablice to struktury danych, które przechowują wiele elementów tego samego typu. Deklarujemy je z określoną liczbą elementów, które możemy następnie inicjalizować i modyfikować. Poniżej znajdują się podstawowe operacje na tablicach:

    #include <iostream>

using namespace std;

int main() {
    int arr[5];  // Deklaracja
    int arr[5] = {1, 2, 3, 4, 5};  // Inicjalizacja
    int value = arr[2];  // Indeksowanie
    arr[0] = 10;  // Zmiana wartości

    return 0;
}

    2. Wektory (vector):

    Wektory to dynamiczne tablice, które mogą zmieniać rozmiar w trakcie działania programu. Poniżej pokazano, jak inicjalizować wektor, dodawać i usuwać elementy oraz iterować po jego zawartości:

    #include <iostream>
#include <vector>

using namespace std;

int main() {
    vector vec = {1, 2, 3};  // Inicjalizacja wektora
    vec.push_back(4);  // Dodawanie elementu na końcu
    vec.pop_back();  // Usuwanie ostatniego elementu
    for(int i : vec) { cout << i << " "; }  // Iteracja po elementach wektora i ich wyświetlanie

    return 0;
}

    3. Listy (list):

    Listy to struktury danych, które przechowują elementy w sposób nieuporządkowany. Umożliwiają łatwe dodawanie i usuwanie elementów w dowolnym miejscu. Oto przykłady podstawowych operacji na listach:

    #include <iostream>
#include <list>

using namespace std;

int main() {
    list lst = {1, 2, 3};  // Inicjalizacja listy
    lst.push_back(4);  // Dodawanie elementu na końcu
    lst.pop_front();  // Usuwanie pierwszego elementu
    for(int i : lst) { cout << i << " "; }  // Iteracja po elementach listy i ich wyświetlanie

    return 0;
}

    4. Mapy (map):

    Mapy to struktury danych przechowujące pary klucz-wartość. Każdy klucz jest unikalny, a wartości mogą być dowolnego typu. Oto jak można dodawać, usuwać i iterować po parach klucz-wartość w mapach:

    #include <iostream>
#include <map>

using namespace std;

int main() {
    map mp;
    mp[1] = "One";  // Dodawanie pary klucz-wartość
    mp[2] = "Two";
    for(const auto& pair : mp) { cout << pair.first << ": " << pair.second << endl; }  // Iteracja po parach klucz-wartość

    return 0;
}

    5. Zbiory (set):

    Zbiory to struktury danych przechowujące unikalne elementy. Nie pozwalają na duplikaty i zapewniają szybki dostęp do elementów. Oto przykłady podstawowych operacji na zbiorach:

    #include <iostream>
#include <set>

using namespace std;

int main() {
    set s = {1, 2, 3};  // Inicjalizacja zbioru
    s.insert(4);  // Dodawanie elementu
    s.erase(2);  // Usuwanie elementu
    for(int i : s) { cout << i << " "; }  // Iteracja po elementach zbioru i ich wyświetlanie

    return 0;
}

    6. Napisy (string):

    Napisy to sekwencje znaków, które można manipulować na wiele sposobów. Poniżej znajdują się przykłady operacji na napisach, takich jak łączenie, wyodrębnianie podciągów i wyświetlanie:

    #include <iostream>
#include <string>

using namespace std;

int main() {
    string str = "Hello";
    str += " World";  // Łączenie napisów
    cout << str << endl;  // Wyświetlanie napisu
    string substr = str.substr(0, 5);  // Wyodrębnianie części napisu (od indeksu 0, długość 5)
    cout << substr << endl;

    return 0;
}
