1. Tablice

Tablice w Javie są strukturami danych, które pozwalają na przechowywanie wielu elementów tego samego typu. Oto jak można je zadeklarować, zainicjalizować i używać:

  • Deklaracja:
    int[] tablica;
  • Inicjalizacja:
    tablica = new int[10];
  • Indeksowanie:
    tablica[0] = 5;
  • Operacje podstawowe:
    • Wypełnianie:
      Arrays.fill(tablica, 0);
    • Przeglądanie:
      for (int i : tablica) { System.out.println(i); }
    • Wypełnianie:
      Arrays.fill(tablica, 0);
    • Przeglądanie:
      for (int i : tablica) { System.out.println(i); }


import java.util.Arrays;

public class TabliceExample {
    public static void main(String[] args) {
        // Deklaracja
        int[] tablica;

        // Inicjalizacja
        tablica = new int[10];

        // Indeksowanie
        tablica[0] = 5;

        // Wypelnianie
        Arrays.fill(tablica, 0);

        // Przegladanie
        for (int i : tablica) {
            System.out.println(i);
        }
    }
}

2. Listy (ArrayList)

ArrayList jest dynamiczną strukturą danych, która może zmieniać swoją wielkość w czasie działania programu. Oto podstawowe operacje na ArrayList:

  • Dynamiczna alokacja:
    ArrayList<String> lista = new ArrayList<>();
  • Dodawanie:
    lista.add("element");
  • Usuwanie:
    lista.remove("element");
  • Iteracja:
    • Za pomocą pętli for:
      for (String element : lista) { System.out.println(element); }
    • Za pomocą iteratora:
      Iterator<String> iterator = lista.iterator(); while (iterator.hasNext()) { System.out.println(iterator.next()); }


import java.util.ArrayList;
import java.util.Iterator;

public class ArrayListExample {
    public static void main(String[] args) {
        // Dynamiczna alokacja z uzyciem generykow
        ArrayList<String> lista = new ArrayList<>();

        // Dodawanie
        lista.add("element");

        // Usuwanie (usuń jeśli chcesz zobaczyć wynik)
        lista.remove("element");

        // Iteracja za pomoca petli for-each
        for (String element : lista) {
            System.out.println(element);
        }

        // Iteracja za pomoca iteratora
        Iterator<String> iterator = lista.iterator();
        while (iterator.hasNext()) {
            System.out.println(iterator.next());
        }
    }
}

3. Zbiory (HashSet, TreeSet)

Zbiory w Javie są używane do przechowywania unikalnych elementów. HashSet i TreeSet są różnymi implementacjami:

  • HashSet:
    HashSet<String> set = new HashSet<>();
  • Dodawanie i usuwanie:
    set.add("element");
    set.remove("element");
  • Sprawdzanie unikalności: HashSet zapewnia, że żaden element nie jest powtarzany.
  • TreeSet:
    TreeSet<String> sortedSet = new TreeSet<>();
  • Operacje na TreeSet: TreeSet sortuje elementy automatycznie.
    sortedSet.add("element");


import java.util.HashSet;
import java.util.TreeSet;

public class SetExample {
    public static void main(String[] args) {
        // HashSet z uzyciem generykow
        HashSet<String> set = new HashSet<>();

        // Dodawanie i usuwanie
        set.add("element");
        set.remove("element");

        // Sprawdzanie unikalnosci
        System.out.println("HashSet: " + set);

        // TreeSet
        TreeSet<String> sortedSet = new TreeSet<>();

        // Dodawanie
        sortedSet.add("element");

        // Operacje na TreeSet
        System.out.println("TreeSet: " + sortedSet);
    }
}

4. Mapy (HashMap, TreeMap)

Mapy przechowują dane w formie par klucz-wartość:

  • HashMap:
    HashMap<String, Integer> map = new HashMap<>();
  • Dodawanie i usuwanie:
    map.put("klucz", 10);
    map.remove("klucz");
  • Odczytywanie wartości:
    int value = map.get("klucz");
  • TreeMap:
    TreeMap<String, Integer> sortedMap = new TreeMap<>();
  • Operacje na TreeMap: TreeMap sortuje klucze automatycznie.
    sortedMap.put("klucz", 10);


import java.util.HashMap;
import java.util.TreeMap;

public class MapExample {
    public static void main(String[] args) {
        // HashMap z uzyciem generykow
        HashMap<String, Integer> map = new HashMap<>();

        // Dodawanie i usuwanie
        map.put("klucz", 10);
        map.remove("klucz");

        // Odczytywanie wartosci
        // Sprawdzanie, czy klucz istnieje w mapie
        Integer value = map.get("klucz");
        if (value != null) {
            System.out.println("HashMap Value: " + value);
        } else {
            System.out.println("HashMap: Klucz nie istnieje");
        }

        // TreeMap z uzyciem generykow
        TreeMap<String, Integer> sortedMap = new TreeMap<>();

        // Dodawanie
        sortedMap.put("klucz", 10);

        // Operacje na TreeMap
        System.out.println("TreeMap: " + sortedMap);
    }
}

5. Napisy (String)

String w Javie jest używany do przechowywania i manipulacji tekstem:

  • Tworzenie napisu:
    String str = "Hello, World!";
  • Manipulacja:
    • Łączenie:
      String result = str.concat(" Welcome!");
    • Podział:
      String[] parts = str.split(",");
    • Znajdowanie:
      int index = str.indexOf("World");


public class StringExample {
    public static void main(String[] args) {
        // Tworzenie napisu
        String str = "Hello, World!";

        // Manipulacja
        // Laczenie
        String result = str.concat(" Welcome!");

        // Podzial
        String[] parts = str.split(",");

        // Znajdowanie
        int index = str.indexOf("World");

        // Wyjscie
        System.out.println("Result: " + result);
        System.out.println("Parts: " + String.join(", ", parts));
        System.out.println("Index: " + index);
    }
}
