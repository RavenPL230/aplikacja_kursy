1. Klasy i obiekty

W Javie klasa jest szablonem do tworzenia obiektów. Obiekt jest instancją klasy i zawiera dane oraz metody, które operują na tych danych. Oto przykład definicji klasy oraz tworzenia obiektów:

  • Definicja klasy:
    public class NazwaKlasy { /* atrybuty i metody */ }
  • Tworzenie instancji:
    NazwaKlasy obiekt = new NazwaKlasy();


public class Osoba {
    // Atrybuty klasy
    String imie;
    int wiek;
    
    // Metoda
    public void przedstawSie() {
        System.out.println("Czesc, nazywam sie " + imie + " i mam " + wiek + " lat.");
    }
    
    public static void main(String[] args) {
        // Tworzenie instancji klasy
        Osoba osoba1 = new Osoba();
        osoba1.imie = "Jan";
        osoba1.wiek = 30;
        osoba1.przedstawSie();
    }
}

2. Konstruktor i destruktor

Konstruktor to specjalna metoda, która jest wywoływana podczas tworzenia obiektu i służy do jego inicjalizacji. W Javie destruktor jest realizowany przez garbage collection. Oto przykład:

  • Konstruktor:
    public NazwaKlasy() { /* inicjalizacja */ }


public class Samochod {
    String marka;
    int rokProdukcji;
    
    // Konstruktor
    public Samochod(String marka, int rokProdukcji) {
        this.marka = marka;
        this.rokProdukcji = rokProdukcji;
    }
    
    public void pokazInformacje() {
        System.out.println("Marka: " + marka + ", Rok produkcji: " + rokProdukcji);
    }
    
    public static void main(String[] args) {
        // Tworzenie obiektu z uzyciem konstruktora
        Samochod samochod1 = new Samochod("Toyota", 2020);
        samochod1.pokazInformacje();
    }
}

3. Metody i atrybuty

Metody i atrybuty w klasie mogą mieć różne poziomy dostępu: publiczny, prywatny lub chroniony. Oto przykłady:

  • Publiczne:
    public int atrybut;
    public void metoda() { /* kod */ }
  • Prywatne:
    private int atrybut;
    private void metoda() { /* kod */ }
  • Chronione:
    protected int atrybut;
    protected void metoda() { /* kod */ }


public class KontoBankowe {
    private double saldo;
    
    // Konstruktor
    public KontoBankowe(double saldo) {
        this.saldo = saldo;
    }
    
    // Metoda publiczna
    public void wplac(double kwota) {
        saldo += kwota;
    }
    
    // Metoda publiczna
    public void sprawdzSaldo() {
        System.out.println("Aktualne saldo: " + saldo);
    }
    
    public static void main(String[] args) {
        KontoBankowe konto = new KontoBankowe(1000);
        konto.wplac(500);
        konto.sprawdzSaldo();
    }
}

4. Dziedziczenie

Dziedziczenie pozwala tworzyć nowe klasy na podstawie istniejących klas, odziedziczać ich atrybuty i metody. Java wspiera tylko pojedyncze dziedziczenie klas:

  • Klasa bazowa:
    public class KlasaBazowa { /* atrybuty i metody */ }
  • Klasa pochodna:
    public class KlasaPochodna extends KlasaBazowa { /* dodatkowe atrybuty i metody */ }


public class Zwierze {
    public void wydajDzwiek() {
        System.out.println("Zwierze wydaje dzwiek");
    }
}

public class Pies extends Zwierze {
    @Override
    public void wydajDzwiek() {
        System.out.println("Pies szczeka");
    }
    
    public static void main(String[] args) {
        Pies pies = new Pies();
        pies.wydajDzwiek();
    }
}

5. Polimorfizm

Polimorfizm pozwala na używanie obiektów różnych klas w jednolity sposób. W Javie realizowany jest poprzez metody wirtualne i dynamiczne wiązanie:

  • Metody wirtualne:
    Metody w klasach bazowych, które mogą być nadpisywane w klasach pochodnych.
  • Dynamiczne wiązanie:
    Mechanizm, który pozwala na wywoływanie metod w czasie wykonywania programu, w zależności od rzeczywistego typu obiektu.


/* Plik: Animal.java */
public class Animal {
    public void makeSound() {
        System.out.println("Animal makes a sound");
    }
}

/* Plik: Cat.java */
public class Cat extends Animal {
    @Override
    public void makeSound() {
        System.out.println("Cat meows");
    }
}

/* Plik: PolymorphismExample.java */
public class PolymorphismExample {
    public static void main(String[] args) {
        Animal myCat = new Cat(); // Dynamiczne wiazanie
        myCat.makeSound(); // Wywoluje metode klasy Cat
    }
}

6. Przykład projektu z użyciem klas

Oto prosty projekt pokazujący użycie klas, dziedziczenia i polimorfizmu:

/* Plik: Produkt.java */
public class Produkt {
    String nazwa;
    double cena;
    
    public Produkt(String nazwa, double cena) {
        this.nazwa = nazwa;
        this.cena = cena;
    }
    
    public void pokazInformacje() {
        System.out.println("Produkt: " + nazwa + ", Cena: " + cena);
    }
}

/* Plik: Elektronika.java */
public class Elektronika extends Produkt {
    String marka;
    
    public Elektronika(String nazwa, double cena, String marka) {
        super(nazwa, cena);
        this.marka = marka;
    }
    
    @Override
    public void pokazInformacje() {
        System.out.println("Produkt: " + nazwa + ", Cena: " + cena + ", Marka: " + marka);
    }
}

/* Plik: Main.java */
public class Main {
    public static void main(String[] args) {
        Produkt produkt = new Produkt("Książka", 29.99);
        produkt.pokazInformacje();
        
        Elektronika elektronika = new Elektronika("Smartfon", 799.99, "Samsung");
        elektronika.pokazInformacje();
    }
}
