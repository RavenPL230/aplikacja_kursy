1. Funkcje anonimowe (lambda)

Funkcje lambda w Javie pozwalają na przekazywanie kodu jako argumentów. Oto przykład:



import java.util.function.BiFunction;

public class LambdaExample {
    public static void main(String[] args) {
        // Funkcja lambda do dodawania dwoch liczb
        BiFunction<Integer, Integer, Integer> dodaj = (a, b) -> a + b;

        System.out.println(dodaj.apply(5, 3)); // Wydrukuje: 8
    }
}

2. Interfejsy funkcyjne

Interfejsy funkcyjne są używane w połączeniu z funkcjami lambda. Przykład:

import java.util.function.Function;
import java.util.function.Predicate;

public class FunctionalInterfaceExample {
    public static void main(String[] args) {
        // Funkcja, ktora podnosi liczbe do kwadratu
        Function<Integer, Integer> kwadrat = x -> x * x;
        System.out.println(kwadrat.apply(5)); // Wydrukuje: 25

        // Predykat, ktory sprawdza, czy liczba jest parzysta
        Predicate<Integer> czyParzysta = x -> x % 2 == 0;
        System.out.println(czyParzysta.test(4)); // Wydrukuje: true
    }
}

3. Stream API

Stream API w Javie umożliwia przetwarzanie danych w stylu funkcyjnym. Przykład operacji map() i filter():

import java.util.Arrays;
import java.util.List;
import java.util.stream.Collectors;

public class StreamApiExample {
    public static void main(String[] args) {
        List<Integer> liczby = Arrays.asList(1, 2, 3, 4, 5, 6, 7, 8, 9);

        // Mapowanie: podniesienie kazdej liczby do kwadratu
        List<Integer> kwadraty = liczby.stream()
                                           .map(x -> x * x)
                                           .collect(Collectors.toList());
        System.out.println(kwadraty); // Wydrukuje: [1, 4, 9, 16, 25, 36, 49, 64, 81]

        // Filtrowanie: wybor liczb wiekszych niz 5
        List<Integer> wiekszeNizPieci = liczby.stream()
                                                  .filter(x -> x > 5)
                                                  .collect(Collectors.toList());
        System.out.println(wiekszeNizPieci); // Wydrukuje: [6, 7, 8, 9]
    }
}
