• Kursy Java

1. Co to jest Java?

Java to wszechstronny, obiektowo-zorientowany język programowania, który został stworzony przez firmę Sun Microsystems w 1995 roku. Jest szeroko stosowany w aplikacjach internetowych, desktopowych oraz mobilnych, dzięki swojej zasady „napisz raz, uruchom wszędzie” (write once, run anywhere), która pozwala na uruchamianie aplikacji na różnych platformach bez potrzeby modyfikacji kodu źródłowego.

2. Instalacja Javy i konfiguracja środowiska programistycznego

Aby zainstalować Javę, należy pobrać i zainstalować Java Development Kit (JDK) z oficjalnej strony Oracle. Po zainstalowaniu JDK, można skonfigurować środowisko programistyczne, takie jak IntelliJ IDEA, Eclipse, lub NetBeans. Te narzędzia ułatwiają pisanie, kompilowanie i debugowanie kodu w Javie. Dostępne są również kompilatory Javy do użytku w przeglądarce np: Programiz.

3. Pierwszy program w Javie

Po zainstalowaniu Javy i skonfigurowaniu środowiska, możesz napisać swój pierwszy program. Oto przykład prostego programu "Hello World" w Javie:

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

Po skompilowaniu i uruchomieniu tego programu powinieneś zobaczyć tekst: Hello, World!

4. Podstawowe typy danych w Javie

  • Liczby całkowite:
    • byte: 8-bitowa liczba całkowita, zakres od -128 do 127. Przykład:
      byte b = 100;
    • short: 16-bitowa liczba całkowita, zakres od -32,768 do 32,767. Przykład:
      short s = 10000;
    • int: 32-bitowa liczba całkowita, zakres od -2^31 do 2^31-1. Przykład:
      int i = 123456;
    • long: 64-bitowa liczba całkowita, zakres od -2^63 do 2^63-1. Przykład:
      long l = 123456789012345L;
  • Liczby zmiennoprzecinkowe:
    • float: 32-bitowa liczba zmiennoprzecinkowa, zakres od ±1.4e−45 do ±3.4e38. Przykład:
      float f = 3.14f;
    • double: 64-bitowa liczba zmiennoprzecinkowa, zakres od ±4.9e−324 do ±1.8e308. Przykład:
      double d = 3.141592653589793;
  • Znaki:
    • char: 16-bitowy znak Unicode, zakres od '\u0000' do '\uffff'. Przykład:
      char c = 'A';
  • Napisy:
    • String: Ciąg znaków, który jest używany do przechowywania tekstu. Przykład:
      String str = "Hello, World!";
  • Typ logiczny:
    • boolean: Typ logiczny, który może przyjmować wartości true lub false. Przykład:
      boolean isTrue = true;
  • Typy referencyjne:
    • Object: Typ bazowy dla wszystkich klas w Javie. Przykład:
      Object obj = new Object();

5. Operatory matematyczne, logiczne i bitowe

  • Operatory matematyczne:
    • Dodawanie (+): Służy do dodawania dwóch liczb. Przykład:
      int sum = 5 + 3;
    • Odejmowanie (-): Służy do odejmowania jednej liczby od drugiej. Przykład:
      int difference = 10 - 4;
    • Mnożenie (*): Służy do mnożenia dwóch liczb. Przykład:
      int product = 7 * 6;
    • Dzielenie (/): Służy do dzielenia jednej liczby przez drugą. Przykład:
      int quotient = 8 / 2;
    • Reszta z dzielenia (%): Służy do uzyskania reszty z dzielenia. Przykład:
      int remainder = 10 % 3;
  • Operatory logiczne:
    • AND (&&): Zwraca true, jeśli oba warunki są prawdziwe. Przykład:
      boolean result = (5 > 3) && (8 > 6);
    • OR (||): Zwraca true, jeśli przynajmniej jeden z warunków jest prawdziwy. Przykład:
      boolean result = (5 > 3) || (8 < 6);
    • NOT (!): Zwraca true, jeśli warunek jest fałszywy. Przykład:
      boolean result = !(5 > 3);
    • Równoważność (==): Sprawdza, czy dwie wartości są równe. Przykład:
      boolean isEqual = (5 == 5);
    • Różność (!=): Sprawdza, czy dwie wartości są różne. Przykład:
      boolean isNotEqual = (5 != 3);
  • Operatory bitowe:
    • AND (&): Wykonuje bitową operację AND. Przykład:
      int bitwiseAnd = 12 & 5;
    • OR (|): Wykonuje bitową operację OR. Przykład:
      int bitwiseOr = 12 | 5;
    • XOR (^): Wykonuje bitową operację XOR. Przykład:
      int bitwiseXor = 12 ^ 5;
    • NOT (~): Wykonuje bitową operację NOT. Przykład:
      int bitwiseNot = ~12;
    • Shift w lewo (<<): Przesuwa bity w lewo. Przykład:
      int shiftLeft = 5 << 2;
    • Shift w prawo (>>): Przesuwa bity w prawo. Przykład:
      int shiftRight = 20 >> 2;
