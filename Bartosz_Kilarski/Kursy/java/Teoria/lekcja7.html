1. Otwieranie i zamykanie plików

Do otwierania i zamykania plików w Javie używamy klas takich jak FileReader, FileWriter, BufferedReader i BufferedWriter. Oto przykłady ich użycia:

/* Przykład: Otwieranie i zamykanie plików */
import java.io.*;

public class PracaZPlikami {
    public static void main(String[] args) {
        // Otwieranie pliku do odczytu
        try (BufferedReader bufferedReader = new BufferedReader(new FileReader("plik.txt"))) {
            String linia;
            while ((linia = bufferedReader.readLine()) != null) {
                System.out.println(linia);
            }
        } catch (IOException e) {
            e.printStackTrace();
        }

        // Otwieranie pliku do zapisu
        try (BufferedWriter bufferedWriter = new BufferedWriter(new FileWriter("plik.txt"))) {
            bufferedWriter.write("Przykladowy tekst");
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

W powyższym przykładzie:

  • Plik o nazwie plik.txt jest otwierany do odczytu za pomocą BufferedReader i FileReader. Odczytujemy linie z pliku i wyświetlamy je na konsoli.
  • W drugiej części kodu plik plik.txt jest otwierany do zapisu za pomocą BufferedWriter i FileWriter. Zawartość pliku zostaje nadpisana tekstem "Przykladowy tekst".

2. Odczytywanie i zapisywanie danych

Podstawowe operacje na plikach obejmują odczytywanie i zapisywanie danych. Wykorzystując klasy BufferedReader i BufferedWriter, możemy efektywnie przetwarzać dane tekstowe w plikach.

/* Przykład: Odczytywanie i zapisywanie danych */
import java.io.*;

public class OdczytIZapis {
    public static void main(String[] args) {
        // Odczytywanie danych z pliku
        try (BufferedReader bufferedReader = new BufferedReader(new FileReader("plik.txt"))) {
            String linia;
            while ((linia = bufferedReader.readLine()) != null) {
                System.out.println(linia);
            }
        } catch (IOException e) {
            e.printStackTrace();
        }

        // Zapisywanie danych do pliku
        try (BufferedWriter bufferedWriter = new BufferedWriter(new FileWriter("plik.txt"))) {
            bufferedWriter.write("Nowy tekst");
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

W tym przykładzie:

  • Najpierw otwieramy plik plik.txt do odczytu i wypisujemy jego zawartość na konsolę.
  • Następnie otwieramy ten sam plik do zapisu. Zawartość pliku zostaje zastąpiona tekstem "Nowy tekst".

3. Użycie try-with-resources do zarządzania zasobami

Try-with-resources to konstrukcja w Javie, która automatycznie zamyka zasoby po zakończeniu ich używania. Jest to przydatne do zarządzania plikami, aby upewnić się, że pliki są prawidłowo zamykane po ich użyciu.

/* Przykład: Try-with-resources */
import java.io.*;

public class TryWithResources {
    public static void main(String[] args) {
        try (BufferedReader bufferedReader = new BufferedReader(new FileReader("plik.txt"))) {
            String linia;
            while ((linia = bufferedReader.readLine()) != null) {
                System.out.println(linia);
            }
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

W tym przykładzie:

  • Używamy try-with-resources do otwierania pliku plik.txt w trybie odczytu.
  • Po zakończeniu pracy z plikiem, zasób (pliki) zostaje automatycznie zamknięty, nawet jeśli wystąpi wyjątek.
