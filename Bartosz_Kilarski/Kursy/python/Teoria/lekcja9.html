1. Funkcje anonimowe (lambda): składnia i przykłady użycia

Funkcja lambda w Pythonie to anonimowa funkcja, która może mieć dowolną liczbę argumentów, ale tylko jedno wyrażenie. Składnia funkcji lambda wygląda następująco:

# Przykład funkcji lambda
dodaj = lambda x, y: x + y
print(dodaj(5, 3))  # Wynik: 8

Funkcja lambda w powyższym przykładzie działa jak każda inna funkcja, ale jej deklaracja jest bardziej zwięzła.



2. Funkcje wyższego rzędu: map(), filter(), reduce()

Funkcje wyższego rzędu to funkcje, które mogą przyjmować inne funkcje jako argumenty lub zwracać funkcje. Przykłady takich funkcji to map(), filter() i reduce():

Przykład map():

# Funkcja map() stosuje funkcję do każdego elementu iterowalnego
numbers = [1, 2, 3, 4]
squares = list(map(lambda x: x**2, numbers))
print(squares)  # Wynik: [1, 4, 9, 16]

Przykład filter():

# Funkcja filter() filtruje elementy na podstawie warunku
numbers = [1, 2, 3, 4]
evens = list(filter(lambda x: x % 2 == 0, numbers))
print(evens)  # Wynik: [2, 4]

Przykład reduce():

# Funkcja reduce() redukuje iterowalną do pojedynczej wartości
from functools import reduce

numbers = [1, 2, 3, 4]
sum_of_numbers = reduce(lambda x, y: x + y, numbers)
print(sum_of_numbers)  # Wynik: 10

W powyższym przykładzie, reduce() stopniowo redukuje listę liczb, sumując jej elementy.



3. Porównanie list comprehensions z map() i filter()

List comprehensions oferują bardziej zwięzły sposób na mapowanie i filtrowanie danych w Pythonie w porównaniu do map() i filter(). Oto porównanie:

Przykład list comprehension dla mapowania:

# Użycie list comprehension do podniesienia liczb do kwadratu
numbers = [1, 2, 3, 4]
squares = [x**2 for x in numbers]
print(squares)  # Wynik: [1, 4, 9, 16]

Przykład list comprehension dla filtrowania:

# Użycie list comprehension do filtrowania liczb parzystych
numbers = [1, 2, 3, 4]
evens = [x for x in numbers if x % 2 == 0]
print(evens)  # Wynik: [2, 4]

List comprehensions mogą być szybsze i bardziej zrozumiałe dla krótkich operacji mapowania i filtrowania, podczas gdy map() i filter() mogą być bardziej czytelne dla bardziej złożonych operacji, szczególnie gdy korzystamy z istniejących funkcji.



4. Funkcje wyższego rzędu z własnymi funkcjami

W Pythonie funkcje mogą być przekazywane jako argumenty do innych funkcji. Oto przykład funkcji wyższego rzędu, która przyjmuje funkcję jako argument:

# Funkcja wyższego rzędu
def przyjmij_funkcje(func, x):
    return func(x)

# Funkcja do przekazania
def kwadrat(n):
    return n ** 2

print(przyjmij_funkcje(kwadrat, 5))  # Wynik: 25

Funkcja przyjmij_funkcje() przyjmuje dowolną funkcję i stosuje ją do wartości x. W tym przypadku przekazujemy funkcję kwadrat(), która podnosi liczbę do kwadratu.
