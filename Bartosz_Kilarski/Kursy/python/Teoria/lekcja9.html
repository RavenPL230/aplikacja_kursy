1. Wprowadzenie do zarządzania pamięcią: podstawowe pojęcia

W zarządzaniu pamięcią istotne jest rozumienie podstawowych pojęć związanych z alokacją i zwalnianiem pamięci. W C++ zarządzanie pamięcią obejmuje:

  • Wskaźniki i referencje
  • Dynamiczną alokację pamięci
  • Smart wskaźniki
  • Mechanizmy zarządzania pamięcią w C++11 i późniejszych wersjach

2. Wskaźniki i referencje: definicja, użycie i różnice

Wskaźniki to zmienne przechowujące adresy innych zmiennych, natomiast referencje są aliasami dla zmiennych. Oto przykłady:

#include <iostream>

using namespace std;

int main() {
    int a = 10;
    int* ptr = &a; // wskaźnik
    int& ref = a;  // referencja

    cout << "Wartość a: " << a << endl;
    cout << "Wartość przez wskaźnik: " << *ptr << endl;
    cout << "Wartość przez referencję: " << ref << endl;

    return 0;
}

3. Dynamiczna alokacja pamięci: new, delete

Dynamiczna alokacja pamięci pozwala na przydzielanie pamięci w trakcie działania programu za pomocą operatorów new i delete. Przykład:

#include <iostream>

using namespace std;

int main() {
    int* ptr = new int(5); // alokacja pamięci
    cout << "Wartość: " << *ptr << endl;
    delete ptr; // zwolnienie pamięci

    return 0;
}

4. Smart wskaźniki: std::unique_ptr, std::shared_ptr, std::weak_ptr

Smart wskaźniki zarządzają pamięcią automatycznie, minimalizując ryzyko wycieków pamięci. Oto przykłady:

#include <iostream>
#include <memory>

using namespace std;

int main() {
    unique_ptr uniqPtr = make_unique(10); // std::unique_ptr
    cout << "Wartość std::unique_ptr: " << *uniqPtr << endl;

    shared_ptr sharPtr = make_shared(20); // std::shared_ptr
    cout << "Wartość std::shared_ptr: " << *sharPtr << endl;

    weak_ptr weakPtr = sharPtr; // std::weak_ptr
    if (auto spt = weakPtr.lock()) {
        cout << "Wartość std::weak_ptr: " << *spt << endl;
    }

    return 0;
}

5. Mechanizmy zarządzania pamięcią w C++11 i późniejszych wersjach

C++11 wprowadził nowe mechanizmy zarządzania pamięcią, takie jak smart wskaźniki i funkcje pomocnicze. Przykład użycia:

#include <iostream>
#include <memory>

using namespace std;

int main() {
    // Przykład użycia std::make_unique
    unique_ptr ptr = make_unique(30);
    cout << "Wartość std::make_unique: " << *ptr << endl;

    return 0;
}
