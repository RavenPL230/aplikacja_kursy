• Kursy Python

    1. Importowanie modułów w Pythonie:

    Moduły w Pythonie to pliki zawierające definicje funkcji, klas lub zmiennych, które można importować do innych programów. Aby zaimportować moduł, używamy słowa kluczowego import. Przykład:

    import math
print(math.sqrt(16))

2. Moduł os, sys, math, random:

Moduły os, sys, math i random dostarczają narzędzi do pracy z systemem operacyjnym, operacji matematycznych i losowych.

  • Moduł os: Pozwala na interakcję z systemem plików. Przykład:
    • import os
print(os.getcwd())
  • Moduł sys: Umożliwia dostęp do zmiennych i funkcji systemowych. Przykład:
    • import sys
print(sys.version)
  • Moduł math: Zawiera funkcje matematyczne, takie jak pierwiastki, logarytmy. Przykład:
    • import math
print(math.pi)
  • Moduł random: Służy do generowania liczb losowych. Przykład:
    • import random
print(random.randint(1, 10))

3. Tworzenie własnych modułów:

Moduł można stworzyć, zapisując kod Pythona w pliku z rozszerzeniem .py. Można go potem zaimportować w innym pliku. Przykład:

# Plik: moj_modul.py
def powiedz_hello():
    print("Hello, world!")

# Import w innym pliku
import moj_modul
moj_modul.powiedz_hello()

    4. Używanie pip do instalacji zewnętrznych bibliotek:

    pip to menedżer pakietów Pythona, który pozwala instalować zewnętrzne biblioteki, takie jak numpy, requests, i wiele innych. Instalacja nowych pakietów umożliwia rozszerzenie funkcjonalności Pythona bez potrzeby pisania dodatkowego kodu.

    Kroki instalacji pakietu:

    1. Otwórz wiersz poleceń (CMD) lub terminal. W systemie Windows możesz to zrobić, wyszukując "cmd" w menu Start.
    2. Upewnij się, że masz zainstalowanego Pythona oraz menedżera pip. Wpisz poniższą komendę, aby sprawdzić wersję Pythona: 
      python --version
      Jeśli Python jest poprawnie zainstalowany, wiersz poleceń wyświetli wersję Pythona.
    3. Sprawdź, czy masz zainstalowanego pip, wpisując poniższą komendę: 
      pip --version
      Jeśli pip jest zainstalowany, wiersz poleceń wyświetli jego wersję.
    4. Jeśli pip jest dostępny, możesz zainstalować bibliotekę, np. numpy, za pomocą komendy: 
      pip install numpy

    Korzystanie z zainstalowanej biblioteki:

    Po zainstalowaniu biblioteki możesz ją zaimportować do swojego kodu i korzystać z jej funkcji. Oto przykład, jak używać biblioteki numpy do tworzenia tablicy:

    import numpy as np

# Tworzenie tablicy NumPy
tablica = np.array([1, 2, 3])
print(tablica)

    Uwaga:

    Komendę pip install należy wpisać w terminalu lub wierszu poleceń, a nie w edytorze kodu. Po instalacji, aby użyć biblioteki w swoim kodzie, wystarczy zaimportować ją na początku pliku Pythona za pomocą instrukcji import.
