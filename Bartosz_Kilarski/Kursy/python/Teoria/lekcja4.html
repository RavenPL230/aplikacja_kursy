    1. Definicja funkcji: def

    W Pythonie funkcje definiujemy za pomocą słowa kluczowego def, po którym następuje nazwa funkcji, nawiasy, a następnie dwukropek. Kod funkcji umieszczamy we wcięciu. Przykład:

    def powiedz_hello():
    print("Hello!")

2. Parametry i zwracanie wartości:

Funkcje mogą przyjmować parametry i zwracać wartości za pomocą instrukcji return. Przykład:

def dodaj(a, b):
    return a + b

wynik = dodaj(3, 4)
print(wynik)

3. Zmienne globalne vs lokalne:

Zmienne zdefiniowane wewnątrz funkcji są lokalne i widoczne tylko w jej obrębie. Zmienne globalne mogą być używane w całym programie. Przykład:

globalna = 10

def funkcja():
    lokalna = 5
    print(lokalna)

funkcja()
print(globalna)

4. Domyślne wartości parametrów:

Funkcje mogą mieć parametry z domyślnymi wartościami, które są używane, gdy nie podamy argumentów. Przykład:

def powiedz_hello(imie="świat"):
    print("Hello, " + imie + "!")

powiedz_hello()
powiedz_hello("Bartosz")

5. Rekurencja (np. silnia, ciąg Fibonacciego):

Rekurencja to technika, w której funkcja wywołuje samą siebie. Używa się jej np. do obliczania silni lub ciągu Fibonacciego. Przykład silni:

def silnia(n):
    if n == 0:
        return 1
    else:
        return n * silnia(n - 1)

print(silnia(5))

Przykład ciągu Fibonacciego:

def fibonacci(n):
    if n <= 1:
        return n
    else:
        return fibonacci(n - 1) + fibonacci(n - 2)

print(fibonacci(6))
