1. Co to jest Python?

Python jest łatwym do nauki, wszechstronnym językiem programowania. Ma wydajne wysoko-poziomowe struktury danych i proste ale efektywne podejście do programowania zorientowanego obiektowo. Elegancka składnia Pythona i dynamiczne typowanie, wraz z jego naturą interpretowania, czyni go idealnym językiem do skryptów i szybkiego rozwijania aplikacji w wielu obszarach na większości platform..

2. Instalacja Pythona

Aby zainstalować Pythona, możesz pobrać go ze strony python.org. Wystarczy wybrać odpowiednią wersję dla Twojego systemu operacyjnego. Jednym z popularnych kompilatorów do zainstalowania jest Pycharm. Istnieją również strony internetowe które pozwalają na podobne działanie, lecz działają wolniej np. online-python.com

    3. Pierwszy program w Pythonie

    Po zainstalowaniu Pythona możesz napisać swój pierwszy program. Otwórz terminal lub konsolę i wpisz:

    print("Witaj, świecie!")

    Po uruchomieniu tego polecenia powinieneś zobaczyć tekst: Witaj, świecie!

4. Podstawowe typy danych

  • Liczby:
    • Całkowite (int): Liczby całkowite, bez części dziesiętnej. Przykład:
      x = 5
      y = -3
      z = 42
    • Zmiennoprzecinkowe (float): Liczby z częścią dziesiętną. Przykład:
      a = 3.14
      b = -0.001
      c = 2.0
    • Zespolone (complex): Liczby zawierające część rzeczywistą i urojoną. Przykład:
      d = 1 + 2j
  • Napisy (string):
    • Definicja: Ciągi znaków tekstowych, które są otoczone pojedynczymi (') lub podwójnymi (") cudzysłowami. Przykład:
      name = "Alice"
      greeting = 'Hello, World!'
    • Operacje na napisach: Można łączyć napisy (+), powielać (*), lub uzyskać dostęp do ich części poprzez indeksowanie ([]). Przykład:
      greeting = "Hello, " + "World!";
      print(greeting);
  • Listy (Lists):
    • Definicja: Kolekcje uporządkowanych elementów, które mogą być różnych typów. Przykład:
      numbers = [1, 2, 3, 4, 5]
      mixed_list = [1, 'apple', 3.14, True]
  • Krotki (Tuples):
    • Definicja: Kolekcje uporządkowanych elementów, które są niemutowalne (nie można ich zmieniać). Przykład:
      point = (1, 2)
      colors = ('red', 'green', 'blue')
  • Zbiory (Sets):
    • Definicja: Nieuporządkowane kolekcje unikalnych elementów. Przykład:
      unique_numbers = {1, 2, 3}
      letters = {'a', 'b', 'c'}
  • Słowniki (Dictionaries):
    • Definicja: Nieuporządkowane kolekcje par klucz-wartość. Przykład:
      person = {'name': 'Alice', 'age': 25}
      scores = {1: 'one', 2: 'two'}
  • Typ logiczny (Boolean):
    • Definicja: Typy logiczne reprezentujące prawdę i fałsz. Przykład:
      is_valid = True
      is_empty = False
  • Typ None:
    • Definicja: Reprezentuje brak wartości. Przykład:
      data = None

5. Operatory matematyczne i logiczne

  • Operatory matematyczne:
    • Dodawanie (+): Służy do dodawania dwóch liczb. Przykład:
      sum = 5 + 3
    • Odejmowanie (-): Służy do odejmowania jednej liczby od drugiej. Przykład:
      difference = 10 - 4
    • Mnożenie (*): Służy do mnożenia dwóch liczb. Przykład:
      product = 7 * 6
    • Dzielenie (/): Służy do dzielenia jednej liczby przez drugą. Przykład:
      quotient = 8 / 2
    • Dzielenie całkowite (//): Służy do dzielenia z zaokrągleniem w dół do najbliższej liczby całkowitej. Przykład:
      floor_division = 9 // 4
    • Reszta z dzielenia (%): Służy do uzyskania reszty z dzielenia. Przykład:
      remainder = 10 % 3
    • Potęgowanie (**): Służy do podnoszenia liczby do określonej potęgi. Przykład:
      power = 2 ** 3
  • Operatory logiczne:
    • AND (i): Zwraca True, jeśli oba warunki są prawdziwe. Przykład:
      result = (5 > 3) and (8 > 6)
    • OR (lub): Zwraca True, jeśli przynajmniej jeden z warunków jest prawdziwy. Przykład:
      result = (5 > 3) or (8 < 6)
    • NOT (nie): Zwraca True, jeśli warunek jest fałszywy. Przykład:
      result = not (5 > 3)
    • Równoważność (==): Sprawdza, czy dwie wartości są równe. Przykład:
      is_equal = (5 == 5)
    • Różność (!=): Sprawdza, czy dwie wartości są różne. Przykład:
      is_not_equal = (5 != 3)
    • Większy niż (>), Mniejszy niż (<), Większy lub równy (>=), Mniejszy lub równy (<=): Służy do porównań między wartościami. Przykład:
      greater_than = (5 > 3)
      less_than = (3 < 5)
      greater_or_equal = (5 >= 5)
      less_or_equal = (3 <= 5)
