    1. Definicja klasy i obiektu:

    W Pythonie klasy definiujemy za pomocą słowa kluczowego class. Klasa jest szablonem do tworzenia obiektów, a obiekt jest instancją klasy. Przykład:

    class Osoba:
    def __init__(self, imie, wiek):
        self.imie = imie
        self.wiek = wiek

    def przedstaw_sie(self):
        return f"Cześć, jestem {self.imie} i mam {self.wiek} lat."

# Tworzenie obiektu
osoba = Osoba("Anna", 30)
print(osoba.przedstaw_sie())

2.Inicjalizacja obiektów: metoda __init__:

Metoda __init__ jest konstruktorem, który jest wywoływany podczas tworzenia nowego obiektu. Służy do inicjalizacji atrybutów obiektu. Przykład:

class Samochod:
    def __init__(self, marka, model):
        self.marka = marka
        self.model = model

    def opis(self):
        return f"Samochód: {self.marka} {self.model}"

# Tworzenie obiektu
auto = Samochod("Toyota", "Corolla")
print(auto.opis())

3. Metody i atrybuty:

Metody to funkcje zdefiniowane w klasie, które operują na jej atrybutach. Atrybuty to zmienne związane z obiektem. Przykład:

class Kula:
    def __init__(self, promien):
        self.promien = promien

    def objetosc(self):
        return (4/3) * 3.14 * (self.promien ** 3)

# Tworzenie obiektu
kula = Kula(5)
print(kula.objetosc())

4. Dziedziczenie klas:

Dziedziczenie pozwala na tworzenie nowych klas na podstawie istniejących. Nowa klasa dziedziczy atrybuty i metody klasy bazowej. Przykład:

class Zwierze:
    def __init__(self, nazwa):
        self.nazwa = nazwa

    def dzwiek(self):
        return "Zwierzę wydaje dźwięk."

class Pies(Zwierze):
    def dzwiek(self):
        return "Hau hau!"

# Tworzenie obiektu
pies = Pies("Burek")
print(pies.dzwiek())

Dodatkowy przykład projektu z użyciem klas:

Oto prosty projekt wykorzystujący klasy i dziedziczenie:

class Uczen:
    def __init__(self, imie, klasa):
        self.imie = imie
        self.klasa = klasa

    def przedstaw_sie(self):
        return f"Uczeń: {self.imie}, Klasa: {self.klasa}"

class Student(Uczen):
    def __init__(self, imie, klasa, nr_indeksu):
        super().__init__(imie, klasa)
        self.nr_indeksu = nr_indeksu

    def przedstaw_sie(self):
        return f"Student: {self.imie}, Klasa: {self.klasa}, Nr indeksu: {self.nr_indeksu}"

# Tworzenie obiektu
student = Student("Marek", "2B", "12345")
print(student.przedstaw_sie())
