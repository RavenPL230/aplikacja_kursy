    1. Instrukcja if, elif, else:

    Instrukcja if pozwala na wykonanie kodu, jeśli spełniony jest określony warunek. Instrukcja elif (else if) umożliwia sprawdzenie kolejnych warunków, a else wykonuje kod, jeśli żaden z wcześniejszych warunków nie został spełniony. Przykład:

    x = int(input('podaj liczbe'));
if x > 0:
    print("Liczba dodatnia")
elif x == 0:
    print("Zero")
else:
    print("Liczba ujemna")

2.Pętla for i while:

Pętla for jest używana do iteracji po elementach sekwencji, takich jak lista lub ciąg znaków. Pętla while wykonuje blok kodu, dopóki warunek logiczny jest spełniony. Przykłady:

for:
for i in range(5):
    print(i)
while:
i = 0
while i < 5:
    print(i)
    i += 1

3. Wykorzystanie break, continue, i else w pętlach:

Instrukcja break przerywa pętlę, continue przechodzi do następnej iteracji, a else działa, gdy pętla zakończy się normalnie, bez użycia break. Przykład:

continue: 
for i in range(5):
    if i == 3:
        continue
    print(i)
break: 
for i in range(5):
    if i == 3:
        break
    print(i)
else:
    print("Pętla zakończona bez przerwania")

4. List comprehensions i inne uproszczenia pętli:

List comprehensions pozwalają na tworzenie list w zwięzły sposób, w jednej linii kodu. Przykład:

numbers = [x**2 for x in range(5)]]
