• Kursy Python

    1. Otwieranie, zamykanie plików (open(), close()):

    W Pythonie pliki można otworzyć za pomocą funkcji open(). Po zakończeniu pracy z plikiem, należy go zamknąć za pomocą close(). Przykład:

    plik = open("plik.txt", "r")  # Otwiera plik w trybie odczytu
zawartosc = plik.read()  # Odczytuje zawartość pliku
print(zawartosc)
plik.close()  # Zamyka plik

    2. Tryby pracy z plikami: odczyt, zapis, append:

    Podczas otwierania plików można określić tryb pracy:

    • "r" – odczyt
    • "w" – zapis (nadpisuje plik)
    • "a" – dopisywanie (append)

    Przykład zapisu do pliku:

    plik = open("plik.txt", "w")  # Otwiera plik w trybie zapisu
plik.write("Witaj, świecie!")  # Zapisuje tekst do pliku
plik.close()  # Zamyka plik

    3. Odczytywanie i zapisywanie danych do pliku:

    Do odczytu pliku możemy użyć metody read(), a do zapisu write(). Przykład odczytu pliku linia po linii:

    plik = open("plik.txt", "r")  # Otwiera plik w trybie odczytu
for linia in plik:
    print(linia, end="")  # Odczytuje i wyświetla każdą linię
plik.close()  # Zamyka plik

    4. Użycie menedżera kontekstu: with open:

    Python posiada wbudowany mechanizm menedżera kontekstu, który automatycznie zamyka plik po zakończeniu pracy. Przykład:

    with open("plik.txt", "r") as plik:
    zawartosc = plik.read()
    print(zawartosc)  # Plik zostanie automatycznie zamknięty po zakończeniu bloku
